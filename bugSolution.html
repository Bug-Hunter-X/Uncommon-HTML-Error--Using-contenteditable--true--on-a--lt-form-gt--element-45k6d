The solution involves separating editable content from the form submission functionality. Instead of making the entire form editable, we create individual editable elements (e.g., using <div contenteditable="true">) to handle user input and then extract the values for form submission.

<form>
  <label for="name">Name:</label>
  <div id="nameInput" contenteditable="true"></div>
  <button type="submit" onclick="submitForm()">Submit</button>
</form>

<script>
function submitForm() {
  let name = document.getElementById('nameInput').textContent;
  // ... process the name and submit the form ...
}
</script>
